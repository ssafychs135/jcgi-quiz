# 정보처리기사 필기 문제풀이 (jcgi-quiz)

정보처리기사 필기 기출 PDF에서 **1,578문제 (17회차)**를 추출해 브라우저에서 바로
풀 수 있는 정적 문제풀이 사이트입니다. 회차별·과목별·랜덤·오답노트·북마크
모드를 지원하고, 진도와 오답을 localStorage에 자동 저장합니다.

빌드 도구·백엔드 없음. `site/` 폴더를 정적 호스팅하거나 로컬 파이썬 서버로 띄우면
바로 동작합니다.

> 데이터 출처:
> - **공식 8회차** (2020년 1·2회 통합 ~ 2022년 2회) — comcbt.com 정리 학습/교사/해설집 3종 패키지, 정답·해설 검증됨
> - **복원본 9회차** (2023년 1·2·3회, 2024년 1·2·3회, 2025년 1·2·3회) — 시나공 카페 공개 복원본, 정답표만 포함 · 정답이 일부 부정확할 수 있음
>
> 33개 원본 PDF가 함께 포함되어 있어 시험지를 바로 받아볼 수 있습니다 — [출처·라이선스](#출처와-라이선스) 참고.

## 빠르게 시작하기

```bash
cd "$(dirname "$0")"   # 이 README가 있는 폴더로
python3 -m http.server 8000 -d site
# 그러고 나서 브라우저에서 열기:  http://localhost:8000
```

> 그냥 `site/index.html`을 더블클릭해서 열어도 작동하지만,
> 브라우저 보안 정책상 `fetch("data/questions.json")`이 거부될 수 있어
> 위처럼 로컬 서버로 띄우는 걸 권장합니다.

## 주요 기능

- 🎯 **5가지 학습 모드** — 랜덤(10/30/50/100), 회차별(17회), 과목별(5과목), 오답노트, 북마크
- 📋 **공식·복원본 회차 분리 표시** — 공식 검증 8회차와 시중 복원본 9회차를 명확히 구분, 복원본 문제에는 ⚠ 배지로 신뢰도 안내
- 📚 **원본 문제집 페이지** — 17회차 PDF(공식: 학생용·교사용·해설집 / 복원본: 시험지) 한눈에 보기와 다운로드
- 🔗 **문제 ↔ 출처 PDF 직링크** — 각 문제 헤더와 해설 박스에서 원본 시험지/해설집을 새 탭으로 즉시 열기
- 🆕 **아직 안 푼 문제만** 골라서 30개씩 — 진도 관리에 유용
- 📝 **오답 자동 누적** — 틀린 문제는 오답노트에 들어가고, 다시 맞히면 자동 제거
- ⭐ **북마크** — 헷갈리는 문제 즐겨찾기
- 💡 **해설 자동 표시** — 정답 확인 직후 PDF 해설집 본문이 함께 표시 (E로 토글)
- ⚠ **그림/표 의존 문제 자동 플래그** — 텍스트만으로 풀기 어려운 문제 125개 경고 표시 (원본 시험지 링크 동봉)
- 📊 **과목별 / 회차별 정답률 막대 차트**
- 💾 **localStorage 영속** — 진도와 오답이 브라우저에 자동 저장
- 📥 **JSON 백업·복원** — 다른 기기로 진도 이전 가능

## 키보드 단축키

| 키 | 동작 |
|---|---|
| `1` ~ `4` | 보기 선택 |
| `Enter` | 답 제출 (선택 후) / 다음 문제 (제출 후) |
| `→` / `←` | 다음 / 이전 문제 |
| `B` | 북마크 토글 |
| `E` | 해설 보이기·숨기기 |

## 문제집을 다시 추출하려면

PDF가 추가/수정되었을 때 추출기를 순서대로 돌리면 `site/data/questions.json`이
갱신됩니다. **공식 추출 먼저, 시나공 추출 나중에** 돌려야 하는데 시나공 스크립트가
기존 JSON에 이어붙이는 방식이기 때문입니다.

```bash
python3 -m pip install pypdf pymupdf
python3 extract_questions.py    # comcbt 8회차 → 733문제
python3 sinagong_extract.py     # 시나공 9회차 추가 → +845문제 (총 1,578)
```

- `extract_questions.py`는 `site/data/pdfs/<회차폴더>/정보처리기사<코드>(학생용|교사용|해설집).pdf` 패턴(comcbt 학습/교사/해설 3종)을 찾습니다.
- `sinagong_extract.py`는 `site/data/pdfs/YYYY_N회_복원본/*.pdf` 단일 파일을 찾고, **마지막 페이지의 정답표**(예: `1.③ 2.③ …`)를 정답 소스로 사용합니다.
- 새 회차를 추가하려면 해당 스크립트 상단의 `EXAMS` 리스트에 항목을 추가하고, 위 명명 규칙대로 PDF를 넣으세요.

출력 예시:
```
· 2020년 1·2회 통합:  94 문제
· 2020년 3회:  90 문제
...
총 733 문제
   · 소프트웨어 설계: 159
   · 소프트웨어 개발: 147
   · 데이터베이스 구축: 143
   · 프로그래밍 언어 활용: 128
   · 정보시스템 구축 관리: 156
```

## 어떻게 동작하는지

추출 파이프라인 (`extract_questions.py`)은 회차별 3종 PDF를 다음과 같이 사용합니다:

1. **학생용 PDF** → 문제 본문과 4지선다 보기 (`①②③④`) — 가장 깨끗한 텍스트
2. **교사용 PDF 마지막 페이지** → 100문제 정답 답표를 압축 형태로 파싱
   (예: `12345678910` + `③③②④④①②④③②`)
3. **해설집 PDF** → 각 문항 뒤 `<문제 해설>` 마커 다음의 해설 본문

세 PDF를 독립적으로 처리한 뒤 문제 번호로 조인합니다. PDF 페이지 경계에서 텍스트가
잘려 4지선다가 모이지 않거나, 보기가 그림/수식 등으로 비어있는 문항은 자동으로 제외됩니다.
그림이나 표를 참조하지만 보기 텍스트는 살아있는 문제는 `needsImage: true`로 플래그되어
사이트에서 경고 배지가 표시됩니다.

## 한계

- 원본 PDF의 그림·표·코드 블록은 텍스트 추출에서 사라집니다. 해당 문제들은
  사이트에서 ⚠ 배지로 표시되며, 본 시험지를 함께 참고하면 도움이 됩니다.
- **복원본 회차(2023~2025년)**는 한국산업인력공단 공식 자료가 아닌, 응시자 증언을
  바탕으로 시중에 정리된 자료입니다. 정답이나 문제 본문이 실제 시험과 다를 수
  있습니다. 사이트에서는 `📋 복원본` 배지로 명확히 구분됩니다.
- 복원본은 해설이 제공되지 않습니다 (정답표만 포함).
- 일부 문제(예: 정답이 그림이거나 보기 자체가 수식·SQL인 경우)는 자동으로 제외됩니다.
  100문항 기준 평균 약 92~94문항이 추출됩니다.
